0. Инициализационное состояние. 243 - произвольная постоянная (она также используется в состоянии 3).
1. Пустое состояние. Для удобства. Смысловой нагрузки не несет. Никогда не вызывается.
2. Контрольная сумма не совпала. Производится инициализация данных значениями по умолчанию. Попадание в это состояние означает, что по какой-то причине произошел сбой оборудования. Переменная TMP24 резервируется для состояния 3.
3. Пересчет контрольной суммы, 243 - произвольная постоянная (она также используется в состоянии 0). Следующее состояние хранится в переменной TMP24.
4. Спящий режим (StandBy). Включен только датчик питания. Постоянное общение с таймерами (состояние 9).
5. Отрисовка экрана. Если на компаратор поступает сигнал, выводится сообщение об ошибке (аналогично для датчиков температуры).
6. Управление реле и индикаторами (если на компаратор приходит сигнал, отключается только реле 2).
7. Обработка кнопок.
8. Запоминание нового значения T3 только после того, как будет отпущена кнопка enter.
9. Обработка программных таймеров. Переменнвая TMP24 содержит номер следующего состояния.
